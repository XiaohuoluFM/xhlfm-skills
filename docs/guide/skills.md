# 技能体系

## 一条播客链路上的 Skill 地图

| 阶段 | 目标 | 典型 Skill 示例 | 常见输出 |
| --- | --- | --- | --- |
| 选题策划 | 把灵感变成可录制主题 | 选题筛选、受众画像、嘉宾提纲、节目大纲 | 选题池、提纲、录制目标 |
| 录制协作 | 提高沟通与录制准备效率 | 录前清单、问题排序、上下文简报 | 录制清单、访谈流程、协作备注 |
| 内容加工 | 降低后期整理成本 | 转写清洗、摘要提炼、标题生成、shownotes 编写 | 精修转写、摘要、标题、shownotes |
| 分发传播 | 让内容适应不同渠道 | 平台简介改写、社媒文案、短内容切片脚本 | 平台文案、社媒贴文、切片脚本 |
| 复盘增长 | 为下一期提供反馈 | 评论聚类、主题复盘、增长观察、选题回流 | 复盘报告、洞察清单、下一期建议 |

## 一个 Skill 至少应该回答的问题

- 这个 Skill 解决的是哪一个具体问题？
- 它适用于哪个播客阶段？
- 它接收什么输入，输出什么结果？
- 它依赖哪些工具、模型、资料或上下文？
- 它的边界是什么，不应该用来做什么？

## 推荐目录结构

```text
skills/
  episode-outline/
    SKILL.md
    examples/
    assets/
    references/
```

## `SKILL.md` 建议包含的内容

```md
# Skill Name

## Purpose
解决什么问题

## When To Use
适用场景

## Inputs
需要哪些输入

## Outputs
会产出什么

## Workflow
建议步骤

## Constraints
边界、限制和注意事项
```

## 建议优先公开的 Skill 类型

- 选题发现与节目策划
- 访谈提纲与嘉宾准备
- shownotes、标题和摘要优化
- 多平台分发改写
- 社媒切片脚本与二次传播
- 内容复盘与增长洞察
