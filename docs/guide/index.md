# 项目概览

小火炉播客 Skills 是一个围绕播客创作、分发与传播建立的公开知识与技能仓库。

这里的重点不是堆积零散 prompt，而是把播客工作流中高频、重复、容易出错的部分抽象成可复用的 Skill，帮助创作者和协作者更快进入状态，也更容易把经验沉淀下来。
